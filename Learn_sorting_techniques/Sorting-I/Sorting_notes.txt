Selection Sort :-
    Compare the first with all others and swap.
    Replace the lowest to the start array value.
    but loop will run from 0 till n-2 , because the last term(n-1) doesn't need sorting after sorting.
    # first loop is for replacement of values,
    # second will be for traversing all the elements.
    Time Complexity - O(N^2) -> Best, Average and Worst Cases.
Bubble Sort :-
    Pushes the maximum to the left using adjacent swapping.
    only compare ajacents and swap if small and repeat until done.
    Time Complexity - O(N^2) -> Average and Worst Cases, Without optimization.
    Best is O(N) when we add flag to optimize the searching with adding a flag.
Insertion Sort :-
    Increase the the size of array per iteration and the sort the current increased value.
    Time Complexity - O(N^2) -> Average and Worst Cases, Without optimization.
                      O(N)   -> Best Case
